# Scenarios: Performance

## 📋 测试信息

- **测试文件**: `src/test/suite/scenarios/scenarios-performance.test.ts`
- **测试目标**: 验证扩展性能表现

## 🧪 测试场景

### 1. 大量规则处理
**说明**: 100+ 规则的性能

**步骤**:
- 加载包含大量规则的源
- 执行同步和生成

**验证**:
- ✅ 同步时间 < 30s
- ✅ 生成时间 < 5s
- ✅ 内存使用合理
- ✅ UI 不卡顿

### 2. 搜索性能
**步骤**:
- 在大量规则中搜索

**验证**:
- ✅ 搜索响应时间 < 500ms
- ✅ 结果准确
- ✅ 支持模糊匹配

### 3. 并行同步
**步骤**:
- 同时同步多个源

**验证**:
- ✅ 使用 Promise.all 并行
- ✅ 总时间接近最慢单源
- ✅ 错误不阻塞其他源

## 🎯 关键验证点

- ✅ 大数据量处理
- ✅ 响应时间
- ✅ 内存占用
- ✅ 并行优化
